# API

- `GET /health` - health check
- `POST /api/logs` - ingest single log
- `POST /api/logs/batch` - ingest multiple logs
- `GET /api/sessions` - list sessions
- `GET /api/sessions/{id}` - get session
- `GET /api/sessions/{id}/metrics` - get metrics
- `GET /api/sessions/{id}/summaries` - list summaries
- `POST /api/sessions/{id}/summaries/generate` - generate summary
